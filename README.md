## 🔥 Pro 版核心功能矩阵

相比原版单一的格式转换脚本，**Pro 版 (index.html)** 引入了完整的闭环管理生态：

1.  **多格式弹性吞吐**：
    * **新书签源**：完美兼容 AI 吐出的标准/非标准 CSV、TXT 文本流，亦可直接导入原生的 `.html` 或 Chrome 导出的 `.json` 书签包。
    * **旧备份追溯**：支持上传历史 `favorites.html` 或 JSON 文件以激活深度交叉对齐比对。
2.  **可视化群组树与在线精细微调**：
    * 可视化展开“大类-小类-站点”三级资产树，支持随时**一键在线修改**分类群组和网站标题。
    * 支持将当前微调后的预览视图直接逆向导出为标准 `.csv` 电子表格。
3.  **工业级差异比对引擎 (Delta Compare)**：
    * 内置 **Levenshtein 字符距离算法** 模糊分析，支持【域名优先】与【完整 URL 文本】双模检索。
    * 配备 **10%~100% 严格度自由微调滑块**。自动识别出：
        * *智能双向配对集合*（支持手工追加或切断绑定）。
        * *旧备份独有漏网之鱼*（支持一键批量挽回录入新库，或打包导出纯文本）。
        * *当前流独有新增资产*（可正向同步备份入旧档）。
4.  **可疑标志链接秒级复活**：
    * 针对 AI 频繁在敏感/内网链接标注的 `[可疑链接已删除]` 占位符，支持通过历史旧备份自动检索同名站点，**一键批量秒级找回有效网址**。支持无匹配项时的手动补齐。
5.  **异步多线程连通性探测 (测活探针)**：
    * 自研基于浏览器的**高并发异步嗅探探针**，极速分拣出网络挂账、超时或 DNS 错误的僵尸死链。
    * 贴心支持 **⚠️ HTTP 降级风险标红提示**。
    * **白名单管理系统**：支持单条/批量加白，测活自动跳过。
    * **防拦截批量唤醒**：提供 `400ms` 延迟梯度，支持一键安全打开多个选中网页进行人工复核，彻底规避浏览器弹窗封锁。
6.  **双重防呆回收站 (Trash Bin)**：
    * 删除的站点不会直接硬销毁，而是暂存入回收站，支持随时后悔恢复，为十年的历史数据提供绝对的安全网。
7.  **100% 纯前端隔离与隐私保护**：
    * 没有任何数据会上传到服务器，所有解析、去重、测活和组装均在你的浏览器内存中“物理隔离”完成。

## 🛠 黄金三步法：AI 联动重构工作流

### 第一步：原料准备
1. 打开浏览器（Edge/Chrome），进入书签管理器，点击 `...` -> `导出收藏夹`，得到最原始的 `favorites.html` 备份。
2. 准备好你常用的 AI 工具（推荐 ChatGPT / Claude / Kimi）。

### 第二步：投喂 AI 核心咒语（Prompt）
将你导出的 `favorites.html` 源码全选复制（书签过多可分批投喂，每次约200条），附带发送以下指令：

```markdown
【角色设定】 你是数据整理专家。请读取我发送的 HTML 书签代码，清洗并重组为 CSV 格式。

【输出格式】 请严格按此格式输出，不要代码块，直接输出文本： 大类,小类,网站标题,备注说明,网络预警,网址

【要求】
1. 分类：根据网址内容智能归类（如：AI工具/技术开发/生活娱乐/影音视听/NAS资产等）。
2. 备注：用一句话简述该网站的作用或标签。
3. 预警：判断该网站在国内是否能直连（标注：国内直连/需翻墙/访问慢）。
4. 去重：如果遇到完全一样的链接，只保留一个。
5. 保留：不要删除任何链接，即使看起来失效了，若由于政策问题无法直视，请在网址处标明 [可疑链接已删除] 并保留完整标题。

明白请回复“请发送数据”。

```

### 第三步：复制转换与下载

1. 双击在现代浏览器中打开由本项目部署的 `index.html`（或访问在线 GitHub Pages 链接）。
2. 在左侧拖入或粘贴 AI 生成的干净数据文本流。
3. （可选）在右侧拖入你最初的 `favorites.html` 激活历史对比、坏链修复等高阶玩法。
4. 点击 **“开始修复”** 预览，查验无误后点击 **“下载 HTML 书签”**。
5. 去浏览器中 `导入书签`，见证犹如艺术品般的船新收藏夹！

## 🛠 技术栈

* **Core**: 原生 HTML5 / ECMAScript 6+ (纯前端原生驱动)
* **UI Framework**: Tailwind CSS (通过 CDN 异步渲染精美深色系极客 UI)
* **Icon Toolkit**: Font Awesome 6.5.0
* **Algorithms**:
* Levenshtein Distance 动态规划矩阵（处理模糊字串匹配）
* AbortController 信号中断控制（处理测活网络超时）

## 🔒 关于隐私与安全

* **零服务器交互**：本工具不设后端，不收集任何日志。
* **敏感信息防范**：由于 AI 可能会接触到您的书签。**温馨提示：** 在把原始 HTML 发送给第三方 AI 之前，建议大概扫一眼，若含有包含 `token=`、`session_id= `或内网密码的链接，请手动抹除后再投喂。
* **请先备份**：在任何大范围导入/清理前，请务必妥善保留好你最原始的那份 `favorites.html`！
